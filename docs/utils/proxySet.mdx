# proxySet

#### `proxySet` util

This utility create a proxy which mimic the native Set behavior. The API is the same as Set API

```js
import { proxySet } from 'valtio/utils'

const state = proxySet([1,2,3])

state.add(4)
state.delete(1)
state.forEach(v => console.log(v)) // --->  2,3,4
```

In can be used inside a `proxy` as well.

```js
import { proxySet } from 'valtio/utils'

const state = proxy({
   count: 1,
   set: proxySet()
})
```
