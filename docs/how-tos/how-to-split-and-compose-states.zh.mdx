# 如何分割和组合状态

## 您可以分割状态

创建具有嵌套对象的状态。

```js
const state = proxy({
  obj1: { a: 1 },
  obj2: { b: 2 },
})
```

然后您可以将状态分割成片段。它们都是代理。

```js
const obj1State = state.obj1
const ojb2State = state.obj2
```

## 您可以组合状态

您可以创建状态然后组合它们。

```js
const obj1State = proxy({ a: 1 })
const obj2State = proxy({ a: 2 })

const state = proxy({
  obj1: obj1State,
  obj2: obj2State,
})
```

这与前面的示例等效。

## 您可以创建循环状态

虽然用例较少，但您可以创建循环结构。

```js
const state = proxy({
  obj: { foo: 3 },
})

state.obj.bar = state.obj // 🤯
```
